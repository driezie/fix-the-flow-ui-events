### Intro

Ik heb verschillende soorten knoppen gebruikt voor mijn UI elementen. Dit zijn de volgende:
- `click`
- `dblclick`
- `mousedown`
- `mouseup`
- `keypress`
- `blur`
- `load`
- `scroll`
- `touchstart`
- `touchend`

> "Als gebruiker wil ik verschillende element animations hebben voor elke knop met een functie."
